## 概要

音声で言語学習するアプリです。母語で話した内容を目標言語に翻訳し、発音を練習できます。

---

## 使い方

### 学習の流れ

```
     ┌─────────────────┐
     │   1 Speak       ├──┐
     │  (Native lang)  │  │
     └────────┬────────┘  │
              │           │
              ↓           │
     ┌────────┴────────┐  │
     │  2 Translate    │  │
     │   (Auto)        │  │
     └────────┬────────┘  │
              │           │
              ↓           │
     ┌────────┴────────┐  │
  ┌──┤  3 Listen       │  │
  │  │ (Target lang)   │  │
  │  └────────┬────────┘  │
  │           │           │
  │           ↓           │
  │  ┌────────┴────────┐  │
  │  │ 4 Pronounce     │  │
  │  │ (Repeat)        │  │
  │  └────────┬────────┘  │
  │           │           │
  │           ↓           │
  │  ┌────────┴────────┐  │
  │  │   5 Judge       │  │
  │  │ (OK/Try again)  │  │
  │  └────────┬────────┘  │
  │           │           │
  │  [Again]  │  [Next]   │
  └───────────┤           │
              └───────────┘
```

**📝 注意:** 母語と目標言語の選択は、学習開始時に表示される画面で行います。

1. **話す**: マイクボタンをタップして母語で話す
2. **翻訳**: 自動的に学習言語へ翻訳
3. **聞く**: 翻訳結果の発音を聞く
4. **発音**: 聞いた通りに発音する
5. **判定**: 類似度80%以上で合格（設定で変更可能）

### 音声コマンド

結果画面で自然に話しかけるとコマンドを認識します：

| コマンド         | 動作   |
| ---------------- | ------ |
| Next / 次        | 次へ   |
| Again / もう一度 | 再試行 |
| Finish / 終わり  | 終了   |

## 練習モード

### シャッフルモード

複数の言語ペアをランダムな順序で練習するモードです。

- **セットアップ**: 2つ以上の言語ペア（例：日→英、日→仏）を追加し、各ペアの問題数を設定
- **練習**: 問題ごとに言語ペアがランダムに切り替わり、通常の発音練習が行われます
- **結果**: 完了後に正答率と言語ペア別の成績が表示されます

**📝 注意:** Free/Coffee/Lunchプランは言語ペア上限3つ。Study以上で無制限です。

### 復習モード

過去に学習したフレーズから出題し、発音練習を行う復習モードです。

- **セットアップ**: 言語ペアと問題数（5/10/20/30/50問、またはプランに応じて無制限）を選択
- **練習**: 学習履歴からフレーズが出題され、お手本の音声を聞いてから発音します
- **操作**: 判定後は音声コマンド（次/もう一度/終わり）で操作できます

### 再学習

履歴画面から個別のフレーズを選んで発音練習をやり直す機能です。

- **操作**: 履歴の詳細画面から「再挑戦」をタップし、マイクボタンで録音→判定
- **用途**: 以前NGだったフレーズを集中的に練習する場合に便利です

## ミニゲーム

ミニゲームは[ミッション](#ミッション)の条件を達成すると解放されます。

### リバーストーク

音声認識した内容を翻訳し、逆再生で聞くチャット形式のゲームです。

- **操作**: マイクボタンで話すと音声認識→翻訳され、チャットバブルとして表示。各メッセージをタップすると逆再生されます
- **録音逆再生**: 自分の声を録音して逆再生することもできます
- **逆再生モード**: テキスト反転（文字単位で反転して読み上げ）と波形反転（音声データそのものを逆再生）の2種類
- **再生速度**: スライダーで調整可能

### 福笑い

日本の伝統遊び「福笑い」と発音練習を組み合わせたゲームです。

- **操作**: お題のフレーズが表示・読み上げされ、発音すると類似度に応じて顔のパーツが配置されます
- **パーツセット**: 3種類（とり/おかめ/ひょっとこ）
- **完了**: 全パーツが配置されるとスコア（平均類似度）が表示されます

**📝 注意:** パーツ数以上の学習履歴が必要です。

### 発音チャレンジ

学習履歴のフレーズが画面上部から落ちてきて、それを発音して判定するゲームです。

- **操作**: 落ちてくるフレーズカードを見て発音→音声認識で判定（OK/NG）
- **一時停止**: 一時停止中はカードをタップして音声を確認できます
- **結果**: 完了後にOK/NG数と平均類似度が表示されます

**📝 注意:** 5件以上の学習履歴が必要です。

### ライフゲーム

コンウェイのライフゲーム（セル・オートマトン）のシミュレータです。

- **操作**: セルをタップして生死を切り替え、再生ボタンでシミュレーション開始
- **調整**: 速度、ぼかし、グリッドサイズ、時間帯（色テーマ）をスライダーで変更可能

## ミッション

条件を達成してミニゲームを解放するシステムです。

| ミッション           | 解放条件        |
| -------------------- | --------------- |
| ライフゲーム         | 初回起動で解放  |
| リバーストーク       | 3日連続起動     |
| 福笑い（とり）       | 総学習回数10回  |
| 発音チャレンジ       | 総学習回数50回  |
| 福笑い（おかめ）     | 総学習回数100回 |
| 福笑い（ひょっとこ） | 総学習回数200回 |

**📝 注意:** Unlimitedプランでは全ミッションが無条件で解放されます。

## 設定画面

### システム設定

| 項目               | 説明                       |
| ------------------ | -------------------------- |
| **カラースキーム** | システム / ライト / ダーク |

**📝 注意:** アプリ表示言語は、iOS設定の「優先する言語」に従います。

### 言語設定

| 項目               | 説明                                                                                            |
| ------------------ | ----------------------------------------------------------------------------------------------- |
| **言語モデル管理** | 音声認識用の言語モデルをダウンロード/削除。オフライン使用時は必須。母語と目標言語もここで選択。 |

### 音声設定

| 項目             | 説明                                                                         |
| ---------------- | ---------------------------------------------------------------------------- |
| **スピーカー**   | 読み上げ音声の種類                                                           |
| **読み上げ速度** | 0.1〜1.0（初心者は遅め推奨）<br>※ Freeプランは0.5〜1.0。Coffee以上で0.1〜1.0 |
| **効果音の音量** | 0%〜100%                                                                     |

### 発音評価設定

| 項目                     | 説明                                               |
| ------------------------ | -------------------------------------------------- |
| **発音判定を有効にする** | 発音評価機能のON/OFF                               |
| **類似度しきい値**       | 50%〜100%（初心者は50〜70%、上級者は80〜100%推奨） |

### 音声コマンド設定

| 項目                          | 説明                                 |
| ----------------------------- | ------------------------------------ |
| **音声コマンド言語**          | コマンド認識に使用する言語           |
| **Next/Again/Finishコマンド** | 各コマンドのカスタマイズ             |
| **コマンド類似度しきい値**    | コマンド認識の最低類似度（0%〜100%） |

### 情報

- **課金プラン**: 現在のプランを確認・変更
- **ミッション**: ミニゲーム解放条件の確認
- **使い方**: アプリの使い方ガイド
- **アプリについて**: バージョン・開発者・ライセンス情報
- **プライバシーポリシー**: プライバシーポリシーを表示

## 学習履歴画面

### 履歴の表示

各履歴には以下の情報が記録されます：

- 日時
- 元のテキスト（母語）
- 翻訳結果（目標言語）
- 発音結果
- 類似度

### 操作

| 操作                | 説明                                       |
| ------------------- | ------------------------------------------ |
| **タップ**          | 履歴の詳細を表示                           |
| **再挑戦**          | 同じ問題をもう一度練習                     |
| **★ボタン**         | お気に入りに登録/解除                      |
| **左スワイプ**      | 異常報告（翻訳・認識の問題を報告）         |
| **右スワイプ**      | 個別の履歴を削除                           |
| **フィルター**      | すべて/★のみ/★なしで絞り込み               |
| **統計情報**        | 成功率・学習回数などをグラフ表示           |
| **全削除**          | メニューから全履歴を削除                   |
| **CSVエクスポート** | 履歴をCSVファイルでエクスポート（Pro以上） |

**📝 お気に入り登録上限:** Free: 10件 / Coffee: 50件 / Lunch以上: 無制限

## トラブルシューティング

| 問題               | 対処法                                                                |
| ------------------ | --------------------------------------------------------------------- |
| 音声認識が動かない | マイク許可を確認 / オンデバイス認識ONの場合は言語モデルをダウンロード |
| 翻訳できない       | 対応言語ペアか確認 / インターネット接続を確認                         |
| 音声が出ない       | デバイスの音量を確認 / 効果音の音量設定を確認                         |
| 判定が厳しい/甘い  | 類似度しきい値を調整（初心者50〜70%、上級者80〜100%）                 |
