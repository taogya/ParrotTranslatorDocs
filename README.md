## 返金ポリシー / Refund Policy

アプリ内課金の返金は、Appleの返金ポリシーに従って処理されます。購入後90日以内であれば、Appleの問題レポートページから返金をリクエストできます。<br>
Refunds for in-app purchases are processed according to Apple's refund policy. Within 90 days of purchase, you can request a refund from Apple's Report a Problem page.

**返金のリクエスト方法:**
1. [Apple問題レポートページ](https://reportaproblem.apple.com/)にアクセス<br>
   Visit [Apple Report a Problem page](https://reportaproblem.apple.com/)<br>
2. Apple IDでサインイン<br>
   Sign in with your Apple ID<br>
3. 該当する購入項目を選択<br>
   Select the relevant purchase item<br>
4. 返金理由を選択して送信<br>
   Choose a refund reason and submit<br>

> [!IMPORTANT]<br>
> 返金の承認はAppleによって判断されます。開発者は返金処理に直接関与できません。<br>
> Refund approval is determined by Apple. The developer cannot directly intervene in the refund process.
